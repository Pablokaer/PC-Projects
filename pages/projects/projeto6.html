Sobre o Projeto

O Hub de Contato Inteligente é uma solução full stack para formulários de contato com validação robusta no frontend e no backend. O sistema processa submissões de forma assíncrona, envia e-mails com templates HTML responsivos via Nodemailer e armazena um histórico das mensagens recebidas.


A proteção contra spam é feita em duas camadas: reCAPTCHA v3 no frontend (sem fricção para o usuário) e rate limiting via Express no backend. Todos os dados são sanitizados antes do processamento para prevenir injeção de código. O endpoint aceita requisições CORS apenas de origens autorizadas.
